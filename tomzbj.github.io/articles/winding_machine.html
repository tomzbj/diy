土绕线机

2009-11-01 by Stavros

本来是个PWM调电机转速的板子, 做多了几块, 于是改造了一下. 用M48的INT0口数圈数, 在电机接轴器上粘了一块磁铁, 用A3144检测. 电机是300转的减速电机.

两个按键, 其中一个按了转100圈, 另一个键是10圈.

用来绕变压器很方便.
